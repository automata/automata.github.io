Software

LSDj

Get the full version by donation.

  • LSDj releases and tools: https://www.littlesounddj.com/lsd/latest/

LSDj Patcher

Tool to add your kits to rom and tweak it. After that you can save it using EMS QART.

wget https://github.com/jkotlinski/lsdpatch/releases/download/v1.1.6/LSDPatcherv1_1_6.jar
java -jar LSDPatcherv1_1_6.jar

LSDj Manager

Tool to load your sav files.

sudo apt-get install libxrender1:i386 libxtst6:i386 libxi6:i386
wget https://github.com/jkotlinski/lsdmanager/releases/download/v1.0/LSDManager10.jar
java -jar LSDManager10.jar

EMS QART

Tool to read/write roms/sav files from/to EMS SmartCart.

wget https://github.com/rbino/ems-qart/releases/download/v0.9.4/ems-qart_0.9.4-1_amd64.deb
sudo dpkg -i ems-qart_0.9.4-1_amd64.deb
sudo apt-get install -f

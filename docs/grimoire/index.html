Assorted snippets of code that were useful in the past and can be even more in the future.

Linux / Bash

Concatenate two or more PDF files into one:

pdfunite one.pdf two.pdf out.pdf

Convert two or more PNG files into one PDF:

convert one.png two.png out.pdf

Mount secure folder to remote folder:

sudo apt install sshfs
mkdir -p $HOME/folder
sshfs -o idmap=user user@remote-host:/home/user/folder $HOME/folder
